QUESTION : Compute $P(A|B)$, if $P(B) = 0.5$ and $P (A \cap B) = 0.32.

SOLUTION : \[P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)} = \frac{0.5 \cdot 0.64}{0.5} = 0.64\]
